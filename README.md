-- Book Scraper --

A Python web scraping project that extracts book titles and prices from a website and saves them to a CSV file.

## 🛠 Tools Used
- Python
- BeautifulSoup
- Pandas
- Requests

## ⚙️ How It Works
1. Connects to books.toscrape.com
2. Finds all book listings on the page
3. Extracts title and price from each book
4. Saves the data into a books.csv file

## ▶️ How to Run
pip install requests beautifulsoup4 pandas
python scraper.py

## 📊 Output
A CSV file with 20 books containing their titles and prices.
